# Mebro

## Trust Me Bro? Let's Check.

> **[Try it live at mebro.app â†’](https://mebro.app)**

**Mebro** is an AI-powered fact-checking application that helps users verify claims, quotes, and information they encounter online. Because "trust me bro" isn't a source.

### Features

- **Instant Fact Checking** - Paste any claim and get AI-powered verification
- **Tone Selection** - Choose your response style:
  - **Cordial** - Friendly, conversational fact-checks
  - **Academic** - Scholarly, citation-heavy analysis
  - **Brutal** - No-nonsense, direct verdicts
- **Shareable Links** - Generate links to share fact-check results
- **Recent Fact Checks** - Browse community-submitted claims
- **Credit System** - Fair usage with credits (5/5 shown)

### How It Works

1. Paste a claim, quote, or something you saw online
2. Select your preferred tone
3. Click "Fact Check This"
4. Get AI-powered analysis with sources

### Tech Stack

- TypeScript
- React / Next.js
- AI/LLM Integration for fact verification
- Warm, accessible UI design
